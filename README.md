# crop_advisor_system
AI Crop Advisor Chatbot is a Flask-based intelligent agriculture assistant that helps farmers make informed decisions about crop selection, weather conditions, and farming practices. The application integrates Google Gemini AI to provide personalized agricultural guidance and recommendations. Users can register, log in, and interact with the chatbot to receive crop-related advice, weather updates, and farming information. The system uses a local SQLite database for user management and chat history storage. The responsive interface is designed to be simple and user-friendly, making agricultural knowledge easily accessible. This project demonstrates the integration of Generative AI, weather APIs, database management, and modern web development technologies to support smart farming solutions.
